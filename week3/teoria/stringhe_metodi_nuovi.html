Metodi nuovi delle stringhe

Abbiamo gia isto metodi nuovi come: length (lunghezza), slice, substring, substr (tagliare le stringhe), toUpperCase e toLowerCase (per renderle maiuscole o minuscole), trim (pulire dagli spazi bianchi)

Metodi di controllo sulle stringhe: mi permettono di controllare se una stringa inizia, finisce o contiene determiunati caratteri.

Controllo se una stringa inizia con determinati caratteri -> metodo startsWith

Constrollo se una stringa finisce con determinati caratteri -> endsWith

Controllare se una stringa contiene una determinata parola -> includes
