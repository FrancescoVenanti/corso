set interval e set timeout

Metodi che ci permettono di gestire attivita' differite nel tempo, faremo riferimento a timer (in millisecondi)

Metodo Descrizione
setInterval() Esegue una funzione in base a un intervallo di tempo specificato
setTimeout() Esegue una funzione dopo un intervallo di tempo specificato
clearInterval() Azzera il timer creato con setInterval()
clearTimeout() Azzera il timer creato con setTimeout
