Inserisco codice inline

Posso inserire direttamente le ist5ruzioni javascript in un elemento HTML, assegnandolo ad un attributo che rappresenta un evento

Altro approccio per inserire codice inline, ma è valido solo per i link

Inserisco blocchi di codice

Pro:nel caso di poco codice javascript metto in relazione subito html e js

Contro: nel caso il codice sia minimamente complesso (variabil, funzioni, oggetti) non risulta comodo

Tag head o tag body primo ordine di considerazione -> performance;
secondo ordine di considerazioni -> che il rag su cui js interagisce sia già stato parsato (letto e quindi disponibile)
-> questo spiega perche tante volte troviamo gli script in fonto alla pagina prima della chiusura di <body>

Inserisco javascript esterno

Approccio più consigliato. consiste nel collegare a una pagina html uno script presente in un file esterno
Il grande vantaggio di questo approccio è la separazione tra struttura e programmazione, tra cos'è un tag e cosa deve fare.

Oggetto console

La console è uno strumento fondamentale per scrivere codice javascript (per qualsiasi liunguaggio di programmazione)

La console ci permette di fare il debug, cioè ci permette di controllare il flusso del nostro codice

Spesso la console si usa per tenere traccia dell'evoluzione del programma o verificare dei valori
