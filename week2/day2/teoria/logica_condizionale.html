Logica Condizionale

Qualunque programma e' una sequenza di istruzioni che vengono eseguite dal mio engine js

Le nostre istruzioni (tutto cio che scriviamo all'interno di uno script) sono un flusso di comandi che stabilisce il comportamento del sistema (nel mio caso pagina web)

per portarlo da uno stato inziale a uno stato finale. In altre parole, il programma ci permette di descrivere un algoritmo tramite cui possiamo raggiungere un certo risultato

Per ottenere un risultato avremo bisono di alcune istruzioni che ci aiutano a stabilire il flusso di esecuzione del nostro programma.

Blocchi di codice

Piu istruzioni possono essere raggruppare in blocchi che fungono come una singola istruzione.
I blocchi di codice sono importanti quando evidenziamo che un gruppo di istruzioni deve essere eseguito interamente.
tipicamente nella logica condizionale eseguiamo dei blocchi di codice al seguito di una condizione
un blocco di codice e' racchiuso tra parentesi graffe

la Logica condizionale e' una categoria particolare di istruzioni che consentono di eseguire di eseguire blocchi di codice alternativi in base alla verifica di una condizione. Lo script in un certo senso decide se eseguire un blocco di codice o un altro, la decisione sara basata sulla verifica della condizione
js prevede due istruzioni condizionali

  • if
  • switch

if prevete tre forme

  • if semplice
  • if con alternativa "if...else"
  • if a cascata "if...else if...else if...else"

If semplice

La forma if semplice esegue un blocco di codice solo se una condizione e' vera
se la condizione non e' vera il blocco di codice non viene eseguito

Ecco lo schema sintattico

if...else

in questa forma viene eseguito un blocco di codice oppure un altro in base al valore della condizione

Ecco lo schema sintattico

if a cascata

if a cascata mette a disposizione piu alternative di esecuzione e quindi verificare piu condizioni
