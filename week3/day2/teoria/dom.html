Dom

Document object model e' una proprieta' dell'oggetto window e rappresenta tutta la struttura html caricata nella mia pagina ha una struttura a oggetto. Il dom fornisce una rappresentazione gerarchica di oggetti, che viene spesso chiamata Dom tree assomiglia ad un albero genealogico. Questo albero di oggetti e' accessibile tramite javascript, infatti possiamo fare:

  • Leggere gli elementi
  • modificare la struttura degli elementi
  • modificare il contenuto dei mei elementi (tag)
  • modificare lo stile degli elementi

Il DOM si presenta come un albero con una serie di nodi che hanno diversi tipi

Tipo di nodo Descrizione Esempio
Documento La radice del mio albero, normalmente solo uno, ma se uso il tag frame (esempio youtube, google map etc.) ci sono piu nodi di tipo documento in una pagina <html>
elemento Individeua un tag html <body> <div> <p>
attributo un nodo di tipo attributo corrisponde ad un attributo di un tag html attributo src del tag img, attributo href del tag a
testo contenuto testuale di un tag lorem ipsum
Nelle specifiche W3C esistono altri tipi di nodi, es: commento, document type, ma li trascuriamo perche non faremo mai manipolazioni js su questi tipi di nodo

Manipolare il dom via js

  1. selezionare un elemento del dom
  2. modificare lo "style" di un elemento
  3. modificare le classidi un elemento
  4. Modificare il contenuto di un elemento
  5. Creare e inserire e rimuovere elementi dentro al dom
