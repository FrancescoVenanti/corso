Array

L'array e' una variabile multidimensionale. Dentro una variabile posso inserire un solo valore, dentro a un array posso inserire piu' valori.

Dentro a un array incontriamo strutture di dati omogenee

Dentro ad un array possiamo conservare qualunque tipo di dato (primitivi o oggetti)

Gli array sono degli elenchi / liste dentro cui salvare diversi tipi di dati

Dentro ad un array posso associare piu valori ad un unico nome di variabile

L'uso degli array ci evita di definire piu variabili e semplifica lo svolgimento dell'operazione
